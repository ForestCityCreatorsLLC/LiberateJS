# ⚡ LiberateJS

> **LiberateJS** is a standalone developer utility designed to liberate web applications from proprietary ecosystems and seamlessly migrate them to clean, standalone React and Node.js codebases.

[![License: MIT](https://img.shields.io/badge/License-MIT-emerald.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.io/badge/version-1.0.0--beta-teal.svg)](#)

---

## 🌟 Key Features

*   **Zero-Dependency Migrations**: Translates proprietary syntax, templates, and patterns into standard React components.
*   **Safety-First Engine**: Built-in verification steps to preserve logic integrity and prevent regressions.
*   **Modern Build Tooling**: Automatically provisions your project with Vite, TailwindCSS, and ESLint.
*   **Instant Dev Experience**: Spawns a local development environment instantly, getting you to `npm run dev` in seconds.

---

## 🚀 Quick Start

Ensure you have [Node.js](https://nodejs.org/) installed (v18+ recommended).

### Install CLI
Run the utility directly using `npx`:

```bash
npx liberatejs@latest --src ./proprietary-app --dest ./liberated-react-app
```

### Commands

*   `init`: Initialize a migration config file.
*   `analyze`: Scans the source folder and lists dependencies to migrate.
*   `run`: Executes the migration and outputs a fresh React app.
